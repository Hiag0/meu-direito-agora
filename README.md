# Meu Direito Agora

Plataforma informativa em **Next.js** para orientar cidadãos sobre:
- benefícios previdenciários (ex.: BPC/LOAS, auxílio por incapacidade temporária, auxílio-acidente e salário-maternidade);
- canais oficiais de saúde e ouvidoria;
- triagem inicial com checklist de documentos e próximos passos.

## Aviso importante
O sistema é **informativo e educativo**.  
Não substitui advogado(a), Defensoria Pública ou decisão do órgão competente.

## Tecnologias
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS

## Como executar localmente
```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.
